org.isarnproject.collections.mixmaps.ordered

OrderedMapLike

trait OrderedMapLike[K, +V, +IN <: INodeMap[K, V], +M <: OrderedMapLike[K, V, IN, M] with SortedMap[K, V]] extends NodeMap[K, V] with OrderedLike[K, IN, M] with SortedMapLike[K, V, M]

An inheritable (and mixable) trait representing Ordered Map functionality that is backed by a Red/Black tree implemenation.

K

The key type

V

The value type

IN

The internal node type of the underlying R/B tree subclass

M

The map self-type of the concrete map subclass
